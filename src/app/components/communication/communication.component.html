Ways for communication between Components.
1) Service.
2) @Input () + @Output () with Eventemitter() =>
3) Routing.
4) RXJS(observables, subject)


Service: Singleton Class: only one object.
1) Logic.
2) API / Database Call.
3) Communicate between two Components we need service.

e.g. 1) Component wants to show users from database.
2) Component cannot directly communicate with database.
3) Now we have a API to fetch users.
4) We will use service to fetch data from API.
5) Once data is fetched in service then we inject service to component.
