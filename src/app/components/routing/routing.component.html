Routing Navigation

It is used to go from one html to another html. In angular it is done without reloading the webpage.

    Steps for Routing are:
  1. Giving path in app.routes.ts file
  2. Placing the routerLink="path" attribute in anchor tag or button tags
  3. Remember to include RouterLink module in .component.ts imports. in which u have used routerLink
  4. Placing < router-outlet > tag at a place which is to be changed.

The sidebar is developed using Routing.
